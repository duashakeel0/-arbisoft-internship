

### Prompt 1 — Modern JavaScript Deep Dive

Act as a senior JavaScript engineer and teach me the following concepts from beginner to intermediate level: closures, hoisting, callback functions, Promises, async/await, and ES Modules. For each concept explain:

* What problem it solves
* How it works internally
* Real-world use cases
* Common mistakes
* Interview questions
* A practical code example

Finally, explain the evolution from callbacks → Promises → async/await and compare all three approaches.

### Prompt 2 — React Fundamentals

Act as a React mentor and teach me components, JSX, props, state, event handling, conditional rendering, and list rendering. For each topic:

* Explain the concept
* Show a working example
* Explain the code line by line
* Explain when to use it in real applications
* Highlight common beginner mistakes

Finally, build a small React application that demonstrates props, state, conditional rendering, and list rendering together.

### Prompt 3 — TypeScript for React Developers

Teach me TypeScript fundamentals required for React development. Cover:

* Primitive types
* Arrays and objects
* Function typing
* Interfaces
* Type aliases
* Generics

Compare interfaces vs type aliases, explain when each should be used, and provide practical examples for every concept.

### Prompt 4 — Understanding Vite and Modern Frontend Tooling
